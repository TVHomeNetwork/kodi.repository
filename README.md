# repository.tvhn

## How to install

1.  Download repository archive [repository.tvhn](https://raw.githubusercontent.com/TVHomeNetwork/kodi.repository/master/repository.tvhn/repository.tvhn-1.0.0.zip)
2.  Move it to a directory that Kodi can access
3.  Enable "Unknown Sources" in the settings
4.  Select 'Install from zip file' and find repository archive

Once installed, select "Install from repository", search for "tvhn repository" and install any available addons.
